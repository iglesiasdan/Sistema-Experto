Requerimientos para operar el Sistema

Sistemas Operativos Soportados Windows XP SP2 o superior Ubuntu 8.12 o superior Fedora 9.0 o superior, etc.
Java Jdk 1.6 o superior
Swi-Prolog w32pl583.exe o superior

Ingreso al Sistema

Cargando el sistema se mostrara una figura como la siguiente, además de ser la pantalla de presentación del sistema experto de Amparo Laboral



Al iniciar el sistema se mostrará una pantalla como se muestra en la siguiente figura. Para ingresar al sistema experto, primero hay que autentificarse como usuario o administrador para ello, es necesario un nombre de usuario y su respectiva contraseña.



Luego hacer clic en el botón aceptar


Una vez autentificado, se mostrara la pantalla que vemos en la siguiente figura.



Descripción de los botones


Botón Inicio


Este botón permite regresar al inicio de la aplicación y poder realizar desde un comienzo las consultas al Sistema Experto.

Botón Reporte Requisitos y Criterios


Este botón permite mostrar un documento en formato pdf con todos los criterios utilizados en el Sistema Experto.

Botón Ayuda


Este botón nos muestra la ayuda del Sistema Experto.

Botón Acerca de


Este botón permite mostrar un ventana en la cual se da una pequeña descripción del Sistema Experto además de sus funcionalidades.

En el lado derecho del Sistema Experto se muestra la hora del sistema, el usuario que accedió al sistema y el botón para cerrar sesión.


Para Empezar el procedimiento hacer clic en el botón como se muestra en la figura
